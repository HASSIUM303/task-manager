# Консольный менеджер задач

Простое, но эффективное приложение для управления задачами с интерфейсом командной строки (CLI), написанное на C#.

## Обзор

Консольный менеджер задач — это легковесное приложение, работающее в терминале, которое позволяет пользователям эффективно управлять ежедневными задачами. Приложение предоставляет основные функции управления задачами через интуитивный консольный интерфейс.

## Возможности

- Добавление новых задач с описаниями
- Просмотр всех задач
- Отметка задач как выполненных
- Удаление задач
- Организация задач в разделы
- Постоянное хранение задач
- Простой и чистый интерфейс командной строки

## Требования

### Для запуска готового приложения:
- Windows x64 (для версии из архива tar.gz)

### Для сборки из исходного кода:
- [.NET 6 SDK](https://dotnet.microsoft.com/download) или новее
- Терминал или командная строка

## Установка

### Вариант 1: Загрузка готового приложения
1. Перейдите на страницу релизов: [ссылка на releases]
2. Скачайте архив `.tar.gz` для Windows x64
3. Распакуйте архив в удобное место
4. Запустите исполняемый файл `ConsoleTaskManager.exe`

### Вариант 2: Сборка из исходного кода
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/task-manager.git
   cd task-manager
   ```

2. Перейдите в директорию проекта:
   ```bash
   cd ConsoleTaskManager
   ```

## Сборка проекта

Для сборки проекта выполните:

```bash
dotnet build
```

## Запуск приложения

### Из исходного кода:
Чтобы запустить приложение из исходного кода, используйте:

```bash
dotnet run
```

### Готовое приложение:
Для запуска готового приложения просто выполните дважды клик по файлу `ConsoleTaskManager.exe` или запустите его из командной строки.

## Структура проекта

- `Program.cs` - Главная точка входа и логика приложения
- `SectionAndTasks.cs` - Структуры данных для организации задач по разделам
- `ConsoleTaskManager.csproj` - Файл конфигурации проекта
- `Program.meth.cs` - Дополнительные методы программы

## Использование

После запуска приложения перед вами откроется меню с опциями:

1. Добавить новую задачу
2. Просмотреть все задачи
3. Отметить задачу как выполненную
4. Удалить задачу
5. Выйти из приложения

Следуйте подсказкам на экране, чтобы перемещаться по приложению.
